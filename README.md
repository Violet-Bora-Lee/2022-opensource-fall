2. ls: list
3. cd: change directory

## vim 에디터

```bash
vim abc.txt // abc.txt 파일을 vim 에디터로 열기
```
- 파일을 처음에 열면 명령모드로 진입한다
- 입력 하려면, 입력모드로 진입해야함
  - 키보드에서 i누르기
- 명령모드 진입은 키보드 esc키 누르기
  - 저장하기(write) --> :w
  - 나오기(quit) --> :q
  - 저장하고 나오기 --> :wq
